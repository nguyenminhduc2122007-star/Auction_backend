##Target
Đóng vai một senior software engineer. Bạn hãy dọn dẹp clean hết tất cả các code hiện tại, tập trung và giữ lại duy nhất 2 tính năng đăng ký và đăng nhập. Hãy code lại logic, database và các file code liên quan trong:
- cmd/api
- internal/database
- internal/models
- internal/handler
- internal/service
- internal/middleware
- internal/routes
- internal/config
Tái cấu trúc database, chỉ giữ cấu trúc đơn giản type struct 
Đảm bảo làm đúng những gì tôi bảo, tôi chỉ muốn chạy code trên local, và giữ code đơn giản nhất có thể, và dọn dẹp rác của một fresher, không thêm tính năng nếu tôi không yêu cầu. Sau khi làm, tổng kết lại những gì bạn làm.

Đảm bảo test và code lên local hoạt động ổn định.
